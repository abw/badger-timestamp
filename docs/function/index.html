    public joinTimestamp(ts: Object, config: String): String source 

    import {joinTimestamp} from '@abw/badger-timestamp'

    Function to join the constituent parts of a time stamp into a string

    Params:

    NameTypeAttributeDescription
    ts Object

    object containing year, month, day, and optionally, hours, minutes and seconds.
    ts.year String

    the year as an integer
    ts.month String

    the month as an integer from 1 to 12
    ts.day String

    the month as an integer
    ts.[hours] String

    the hours as an integer from 0 to 23
    ts.[minutes] String

    the minutes as an integer from 0 to 59
    ts.[seconds] String

    the seconds as an integer from 0 to 59
    config String
    • optional

    optional configuration options
    config.joint String
    • optional
    • default: ' '

    joining character between date and time parts
    config.dateJoint String
    • optional
    • default: '-'

    joining character for date segments
    config.timeJoint String
    • optional
    • default: ':'

    joining character for time segments

    Return:

    String

    timestamp string of the form YYYY-MM-DD or YYYY-MM-DD HH:MM:SS
